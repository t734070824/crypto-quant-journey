# 数据目录

此目录包含用于分析和回测的历史市场数据和数据集。

## 目录结构

```
data/
├── raw/              # 交易所和API的原始数据
├── processed/        # 清洗和处理后的数据
├── indicators/       # 预计算的技术指标
└── news/            # 新闻数据和情绪评分
```

## 数据来源

- 加密货币交易所（币安、Coinbase等）
- 新闻API
- 链上数据提供商
- 社交媒体情绪数据

## 重要说明

- 大型数据文件已被git忽略
- 始终记录数据来源和采集日期
- 定期更新数据
- 对数据处理脚本使用版本控制
